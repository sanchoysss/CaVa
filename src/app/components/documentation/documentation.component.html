What is ÇaVa?

ÇaVa is a messenger that allows you communicate with other users. Just type a message and send it to a receiver. Your message will be delivered instantly.

Sign up and login

First of all, you need to sign up or login to start using this application. Use sign up link or login link to sign up or login.

Process of authorization

If user didn't logged in system, he will be redirected to the login page. There he needs to fill the form of authorization. To login he needs to enter his login and password.

To sign up he must click the link that will redirect him to the registration form. There he needs to input his login, name (optional parameter), email and password.

If user won't input all necessary data, form won't let him register (the same will be on login page).

Messaging

After registration/signing in client will send request to the server to log in or sign up the user. If everything ok, server will accept authorization by sending a special token through which server can be able to recognize the user. If something went wrong, server returns an error that will be displayed on client side. Here you can see an example of failed authorization.

When user logged in, he can click on button 'Chats' that will redirect him to the list of his chats.

To open any chat just click on it and you will be able to see your message history with a particular user. Type anything in special input for sending the messages and click the button to send your message.

After clicking the button, client will send this message to the server and server notify receiver that he has a new message. Message, that was sent by user, will be displayed in the next way.

Receiver sees this message in the different way.

Users profile

Click on the button with your name (or login if you didn't enter your name during registration) and then you will be able to see your profile or log out.

On yours profile page you can view different information about your account and change it by pressing the button 'Edit profile'.

Here you can write some additional information about your profile. You can change anything, but all the fields, that where used in registration are required.

If you fill any of links on other social networks, it will be displayed on your profile.
